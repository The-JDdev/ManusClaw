# Deep Research Skill

## Protocol
1. Search broadly with 3+ different query variations using web_search
2. Crawl top 3-5 sources using crawl tool
3. Cross-reference facts across multiple sources
4. Synthesize findings into structured analysis
5. Save results to workspace/research_TOPIC.md

## Query Patterns
- Factual: what is X, X definition
- Comparative: X vs Y comparison 2024
- Latest news: X latest developments
- Technical: X implementation best practices

## Output Format
Always produce: Summary, Key Findings, Sources (with URLs), Confidence Score
