The Hermes Update

What's New in v4.0.0

ManusClaw transforms from a powerful agent into a self-improving ecosystem.

Anti-Jailbreak Identity

Hardcoded identity protection with 20+ jailbreak/injection pattern detection. Intercepts DAN-mode attacks, `<|im_start|>` injections, and forces OS-level resistance before reaching the LLM.

Intelligent Long-Wait

Adaptive 30-minute timeouts for deep-thinking models (DeepSeek R1, o1, Claude 3.7). Smart retry logic prevents API spam while handling transient network errors gracefully.

Persistent Task Queue

SQLite-backed background execution. Send tasks to the background via `/bg`. Priority ordering, crash survival, and auto-resume on the next launch.

Universal LLM Routing

Support for OpenAI, Anthropic, Gemini, Mistral, OpenRouter, and fully offline local inferencing via Ollama, LM Studio, or raw GGUF files. Auto-rotates credentials on rate limits.

DAG Orchestration

Multi-Agent Brain

A MetaGPT-style pipeline orchestrating four specialist roles using Kahn's topological sort. Every role builds on verified output.

1. Product Manager 📝

Writes the structured PRD. Defines the objective, explicit in-scope/out-of-scope boundaries, and measurable acceptance criteria.

2. Architect 🏛️

Produces concrete system design, exact directory trees, technology stacks, and a DAG implementation plan mapping task dependencies.

3. Engineer ⚙️

Delegates to a full Manus instance. Executes the DAG tasks in topological order, writes code, runs it, and self-corrects errors.

4. QA Role 🔬

Runs actual tests based on PRD criteria. Produces a pass/fail QA Report with strict defect tracking to ensure production readiness.

Platform Domination

End-to-end autonomous control over GitHub, Vercel, WordPress, Netlify, Discord, and REST APIs without human intervention.

FTS5 RAG Memory

Auto-Learning Skills

After 5 tool calls, the agent suggests capturing workflows as Markdown skills. Preserves prompt caching seamlessly.

The Quartet

Four Deployment Modes

Run ManusClaw exactly where you need it. From heavy cloud servers to a pocket smartphone.

Standalone CLI

Native terminal experience for Linux, macOS, and Windows. Type manusclaw to wake the persistent shell.

Docker Sandbox

Fully isolated container execution. Ideal for production APIs, CI/CD, and security-sensitive operations.

Termux Native

Run directly on an Android smartphone. Built and tested natively on mobile. No root required.

Desktop GUI

Cross-platform application built with Flet. Terminal-style chat interface with background daemon execution.

# Global Wake-Up Command (after setup)

manusclaw
> AI Environment Active. Type prompts naturally.

# Run a background task
> /bg Build a FastAPI server with auth

# Launch the WebSocket server
manusclaw-server --host 0.0.0.0 --port 8765
